# NoDaysIdle: Cyber Brutalist to Art Deco

An avant-garde portfolio platform that explores the visual transformation from chaotic Cyber Brutalism to elegant Art Deco through interactive, scroll-based experience.

![Project Banner](https://placeholder-image-url.com) *Note: Add a screenshot of the app here*

## üåü Concept

This project posits a unique visual journey: starting from the raw, utilitarian, and chaotic aesthetic of **Cyber Brutalism** and evolving into the refined, geometric, and luxurious style of **Art Deco** as the user interacts with the site. The interface is not just a container but a living artistic statement.

## ‚ú® Features

- **Dynamic Aesthetic Transformation**: Scroll-based transitions that morph the UI from brutalist layouts to Art Deco elegance.
- **Generative Art Gallery**: A curated showcase of algorithmic art pieces that reflect the dual themes.
- **Interactive Shop**: A fully functional e-commerce section featuring themed products.
- **Micro-Animations**: Rich, responsive interactions including hover effects, tilt animations, and custom cursor behaviors.
- **Responsive Design**: A fluid experience across all devices.

## üöÄ Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/salvadalba/nodaysidle-cyberbrutalist-to-art-deco.git
   ```

2. Navigate to the project directory:

   ```bash
   cd nodaysidle-cyberbrutalist-to-art-deco
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

### Development

Start the local development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Build

To create a production build:

```bash
npm run build
```
